    A professional, cryptographically secure password and passphrase generator. Designed for both quick terminal use and deep integration into modern applications.

    • Multi-Mode Generation:
      • Random: Character-based passwords with strict composition rules.
      • Passphrase: Diceware-style generation using an EFF-inspired wordlist.
    • Advanced Control:
      • Set specific minimum counts for lowercase, uppercase, digits, and special characters.
      • Exclude similar-looking characters (i, l, 1, o, 0).

    • Developer-First CLI:
      • Interactive Mode: Guided generation for ease of use.
      • JSON Output: Clean machine-readable output for scripting.
      • Clipboard Integration: Automatically copy results.
      • Security Metrics: Real-time entropy calculation and strength assessment.
    • Universal API: Uses Web Crypto standard (compatible with Node.js and modern browsers).

    ğŸ’» CLI Usage

    Interactive Mode (Recommended) 

    pnpm cli --interactive

    Direct Commands 

    # Generate 5 strong passphrases
    pnpm cli --mode passphrase --length 6 --count 5

    # Strict random password copied to clipboard
    pnpm cli -l 16 --min-special 3 --min-digit 3 --copy

    # JSON output for automation
    pnpm cli --json --verbose

    Available Flags

    Flag Description
    -i, --interactive Run guided setup
    -m, --mode <mode> random or passphrase
    -l, --length <n> Length (chars or words)
    -n, --count <n> Number of results
    -y, --copy Copy first result to clipboard
    -j, --json Output as JSON
    -v, --verbose Show entropy/strength
    -e, --exclude-similar Exclude ambiguous characters
    --min-lower <n> Minimum lowercase required
    --min-special <n> Minimum special chars required
